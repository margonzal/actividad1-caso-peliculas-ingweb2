# Aplicación de Películas - Ingeniería Web II

##  Descripción
Aplicación web desarrollada para la gestión de películas y series mediante una API REST.  
Permite administrar información relacionada con género, director, productora, tipo y media.

---

## Tecnologías utilizadas

### Backend
- Node.js
- Express
- MongoDB Atlas
- Mongoose

### Frontend
- React.js
- Bootstrap
- Axios
- SweetAlert2

---

## Funcionalidades

### API REST
- CRUD de:
  - Géneros
  - Directores
  - Productoras
  - Tipos
  - Medias (películas y series)

### Frontend
- Registro de películas
- Edición y eliminación
- Visualización tipo catálogo
- Detalle de película (sinopsis, género, director, productora)
- Manejo de imágenes mediante URL
